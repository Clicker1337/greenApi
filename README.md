# greenApi
Инструкция для локального запуска

1. Загрузить проект на пк.
2. Открыть при помощи IDE (например VS Code).
3. При помощи консоли, находясь в директории проекта, загрузить необходимые зависимости следующей командой - "yarn".
4. После удачной загрузки зависимости необходимо запустить проект командой "yarn run dev".
5. Нажать в консоли на ссылку.
